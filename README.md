## Cloud Classification Model

A deep learning model is trained to classify cloud types using the Cirrus Cumulus Stratus Nimbus (CCSN) Database, available from Harvard Dataverse. The code and methodology are adapted from the research paper [Cloud type classification using deep learning with cloud images](https://peerj.com/articles/cs-1779/) published by M. Kalkan, M. Guzel, E. Bostanci, K. Acici, and T. Asuroglu.
